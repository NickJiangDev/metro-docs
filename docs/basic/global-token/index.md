# 基础色板

- Metro design System 提供了基础的色板来圈定颜色的适用范围。

- 此处定义的色彩变量为 **“全局变量”** ，在此基础上将会构建“[语义变量](/basic/semantic-token)”。

- 对于全局变量，前端命名描述应当变量本身的特性，例如色彩的类型 Red、Royalblue；色彩的色阶 1、2、3 等等，而不要描述该变量的使用场景，例如 primary、background、font。
